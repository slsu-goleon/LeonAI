# Leon AI On-Device Assistant Model (`leo_1b.task`)

The `leo_1b.task` file is the on-device Large Language Model (LLM) that powers the **Leo AI Assistant** in the GoLeon application. It is based on the Google Gemma architecture (1B Q4 quantized) and is optimized for local execution using MediaPipe on mobile devices.

---

## 🎯 Purpose and Role in the Application

In the GoLeon ecosystem (a comprehensive campus navigation, tour, and information app for Southern Luzon State University - Gumaca Campus), **Leo** serves as a direct chat assistant accessible by students, visitors, admins, and super-admins. 

By running offline on-device, this model ensures that users can access campus information and guidance at any time, even without an active internet connection, while guaranteeing strict user privacy and zero server API costs.
